# 📊 BÁO CÁO PHÁT TRIỂN ỨNG DỤNG FINMATE

## 👤 Thông tin sinh viên
- Họ tên: Hải Nguyễn
- Nhóm: Finmate App
- Nhánh phát triển: feature/hai-newfeature

---

## 📌 Giới thiệu dự án
Finmate là ứng dụng quản lý tài chính cá nhân trên nền tảng Android, giúp người dùng:
- Quản lý thu nhập và chi tiêu
- Theo dõi tài chính theo thời gian
- Hiển thị biểu đồ trực quan
- Phân tích thói quen chi tiêu

---

## 🎯 Mục tiêu phát triển
- Xây dựng hệ thống quản lý tài chính đơn giản, dễ sử dụng
- Tối ưu trải nghiệm người dùng (UI/UX)
- Đảm bảo dữ liệu được lưu trữ an toàn
- Hỗ trợ phân tích tài chính cá nhân

---

## ⚙️ Chức năng chính
- Đăng ký / Đăng nhập người dùng
- Thêm / sửa / xóa giao dịch
- Tính tổng thu nhập
- Hiển thị biểu đồ chi tiêu
- Xem thống kê tài chính

---

## 🛠️ Công nghệ sử dụng
- Ngôn ngữ: Java / Kotlin
- IDE: Android Studio
- Firebase Authentication
- Firebase Realtime Database
- Thư viện biểu đồ: MPAndroidChart

---

## 📊 Phần việc đã thực hiện
- Tạo tài liệu phát triển (docs)
- Nghiên cứu chức năng biểu đồ
- Phân tích dữ liệu tài chính
- Chuẩn bị tích hợp Firebase

---

## 🚀 Hướng phát triển
- Hoàn thiện chức năng biểu đồ
- Tối ưu hiệu năng ứng dụng
- Cải thiện giao diện người dùng
- Thêm chức năng đăng nhập Google

---

## 🐞 Khó khăn gặp phải
- Đồng bộ dữ liệu Firebase
- Xử lý dữ liệu lớn
- Thiết kế UI phù hợp

---

## 💡 Giải pháp
- Sử dụng Firebase để lưu trữ realtime
- Tối ưu thuật toán xử lý dữ liệu
- Tham khảo các UI hiện đại

---

## 📌 Kết luận
Ứng dụng Finmate có tiềm năng trở thành công cụ hỗ trợ quản lý tài chính hiệu quả cho người dùng cá nhân.
